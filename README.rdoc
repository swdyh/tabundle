
= tabundle

A Firefox extention for saving a snapshot of your currently open tabs in an easily-viewed html file and view past tab bundles.

== Installation

http://www.relucks.org/tabundle/tabundle.xpi

=== Archive Installation

	rake xpi

drag and drop tabundle.xpi to Firefox.
